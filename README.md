# Система Гелиос — офлайн карта

Статический HTML/CSS/JS-проект с псевдо-3D визуалом системы Гелиос.

## Реализовано
- Анимация миров по орбитам (индивидуальные скорости из JSON).
- Управление временем: пауза, x1, x5, x20.
- Официальные имперские маршруты из `officialRoutes`.
- Главная магистраль выделяется визуально сильнее (`isMain: true`).
- Маршруты рисуются штрихами по правилу: **1 штрих = 1 парсек**.
- Количество штрихов автоматически зависит от текущей длины маршрута.
- Маршруты пересчитываются в реальном времени по текущим позициям миров.
- Пользовательские маршруты: выбор точки A/B, добавление на карту, штриховка.
- Пользовательские маршруты сохраняются локально в `localStorage`.
- Вкладки в правой панели: Инфо, Мастер, Планеты, Станции, Экономика.
- Заметки по объекту карты + отдельные большие заметки мастера.
- Отдельные текстовые поля (вариант A) для каждой планеты и каждой станции.
- Базовый zoom, pan и выбор сущностей на карте.

## Ключевая архитектурная идея
Маршруты всегда берут координаты через `getEntityWorldPosition(entityId)`, поэтому любая динамика миров автоматически отражается в длинах и штрихах маршрутов.

## Файлы
- `index.html` — разметка интерфейса и панель маршрутов.
- `styles.css` — sci-fi тема и стили карты/маршрутов.
- `app.js` — анимация, рендер, маршруты, localStorage.
- `data/helios.json` — каноничные данные.
- `README.md` — описание проекта.

## Запуск
Откройте через локальный статический сервер, например:
```bash
python -m http.server
```
Затем перейдите в браузере по локальному адресу сервера.
